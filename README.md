Este proyecto es una página web de unna tienda. 

En la carpeta config, está la conexión con la bbdd mongoose.

En la carpeta controllers, hay dos archivos, en los que está la información necesaria para el login y para buscar los productos. 

En la carpeta helpers, hay un archivo que es el que controla los links para poder moverse entre las páginas

En la carpeta models hay dos archivos, con el modelo de esquema que utilizan los productos y el usuario 

En public están las imágenes y las css

En routes, hay tres archivos, y es el conector de la ruta de cada página con los controllers. 

Y el index.js es el archivo prinicpal que conecta bbdd con express, con postman, para poder ver nuestra página bien.
